绘图仪 · 从地图 JSON 生成画面

（1）此页面用于运行绘图仪功能：仅执行 map_device_caller_test，并通过 --map 指定 plot/ 目录下的地图 JSON（CAD 实体数组、像素图或瓦片图格式见 include/maploader.hpp）。
（2）如需在仓库中新增或修改地图 JSON，请返回首页进入「连接终端」页进行文件读写（SSH 读写 plot/*.json）。
（3）像素图 / 瓦片图配色见 include/maploader.hpp 与 color_schema.hpp。CAD 实体地图可用下方「按图层配色」生成带 cad_style 的包装 JSON；亦可手写 { "entities": [...], "cad_style": { ... } }（字段见头文件）。内置中文图层表仍在 C++ 中作默认，可被 JSON 覆盖。

下拉列表来自 /api/files 的 plot_json（与 plot 目录一致，仅 .json）；仅旧接口时可从 plot 筛出 .json。也可手动填写文件名。
