📖 可用变量说明

input.val_rs1 - 源寄存器1的值
input.val_rs2 - 源寄存器2的值
input.imm - 立即数值
input.pc - 当前PC
output.alu_result - 写入目标寄存器的值
output.reg_write - 是否写寄存器 (true/false)
output.mem_write - 是否写内存 (true/false)
output.mem_addr - 内存地址
output.mem_data - 内存数据
output.next_pc - 下一跳PC
output.pc_modified - 是否修改PC (true/false)

💡 提示

- 使用 input 读取输入值
- 使用 output 设置输出
- 支持 JavaScript 语法
- 示例: let result = input.val_rs1 + input.val_rs2;
